# 高校宿舍管理系统 java535

## 项目概述

高校宿舍管理系统是一款基于Spring Boot的前后端分离项目，旨在通过信息化手段提高宿舍管理的效率与准确性。

## 技术栈

- **后端：**
  - Spring Boot
  - Java
- **前端：**
  - Layui
  - jQuery
  - HTML
- **数据库：**
  - MySQL
- **构建工具：**
  - Maven

## 功能模块

### 学生

- 个人信息管理
- 床位选取管理
- 缺勤记录管理
- 保修申请
- 公告信息查看

### 宿管

- 宿舍管理
- 缺勤管理
- 保修管理
- 来访信息管理
- 公告信息管理

### 管理员

- 基本信息管理
- 宿舍信息管理
- 缺勤信息管理
- 报修信息管理
- 来访信息管理
- 公告信息管理

## 系统角色

- 学生
- 宿管
- 管理员
